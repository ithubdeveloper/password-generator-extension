## Suggested Short Description

Generate strong passwords, autofill password fields, and quickly show or hide passwords with an eye toggle.

## Suggested Detailed Description

Password Generator Pro helps you create secure passwords in one click and use them directly in website password fields.

Key features:

- Strong random password generation using secure browser crypto APIs
- Length and character controls (uppercase, lowercase, numbers, symbols)
- One-click copy and autofill for password fields
- In-page quick fill button near password inputs
- Eye toggle support on password fields to show or hide typed passwords
- Works with modern websites and frameworks by dispatching `input` and `change` events
- Settings synced with Chrome storage

Built for speed, security, and better account safety.
